Rock Paper Scissors

15 min conditionals game logic

What We're Building

The classic hand game! You pick rock, paper, or scissors. The computer picks randomly. Who wins?

Game Rules:
  • Rock crushes Scissors
  • Scissors cuts Paper
  • Paper covers Rock
# Sneak peek: A complete game!
import random

choices = ["rock", "paper", "scissors"]
player = "rock"  # Your choice
computer = random.choice(choices)

print(f"You chose: {player}")
print(f"Computer chose: {computer}")

if player == computer:
    print("It's a tie!")
elif (player == "rock" and computer == "scissors") or \
     (player == "scissors" and computer == "paper") or \
     (player == "paper" and computer == "rock"):
    print("You win!")
else:
    print("Computer wins!")

Let's break this down step by step!

Step 1: Making Random Choices

Instead of randint() for numbers, we use random.choice() to pick from a list of options.

import random

# A list holds multiple items in square brackets
choices = ["rock", "paper", "scissors"]

# random.choice picks one item randomly
computer_choice = random.choice(choices)
print(f"Computer picks: {computer_choice}")

# Run multiple times to see different results!
Lists are powerful! They can hold any items: ["apple", "banana"] or [1, 2, 3] or even mixed: ["hello", 42, True]

Step 2: Comparing Text

We compare strings (text) the same way we compare numbers - with ==!

player = "rock"
computer = "scissors"

# Check if they're the same
print(f"Same choice? {player == computer}")

# Check specific values
print(f"Player chose rock? {player == 'rock'}")
print(f"Computer chose paper? {computer == 'paper'}")

Step 3: Detecting a Tie

The easiest case: if both players pick the same thing, it's a tie!

player = "paper"
computer = "paper"

print(f"You: {player}")
print(f"Computer: {computer}")

if player == computer:
    print("It's a tie! Same choice.")
else:
    print("Different choices - someone wins!")

Step 4: Win Conditions (Using 'and')

You win when you pick rock AND computer picks scissors. We combine conditions with and.

player = "rock"
computer = "scissors"

print(f"You: {player}, Computer: {computer}")

# 'and' means BOTH must be true
if player == "rock" and computer == "scissors":
    print("Rock crushes scissors - You win!")

# Try changing player to "paper" and computer to "rock"
Boolean operators:
  • and - Both must be True
  • or - At least one must be True
  • not - Flips True to False

Step 5: All Win Conditions (Using 'or')

There are THREE ways to win. We combine them with or.

player = "scissors"
computer = "paper"

print(f"You: {player}, Computer: {computer}")

# Check all three win conditions
rock_beats_scissors = (player == "rock" and computer == "scissors")
scissors_beats_paper = (player == "scissors" and computer == "paper")
paper_beats_rock = (player == "paper" and computer == "rock")

# 'or' means ANY of these being True counts as a win
if rock_beats_scissors or scissors_beats_paper or paper_beats_rock:
    print("You win!")
else:
    print("You don't win (tie or loss)")

Step 6: The Complete Game!

Let's put it all together with proper if/elif/else logic:

import random

choices = ["rock", "paper", "scissors"]

# Simulate player choice (change this to test!)

print("=" * 30)
String tricks: "=" * 30 creates a line of 30 equals signs. Great for visual formatting!

Step 7: Playing Multiple Rounds

Let's simulate a best-of-5 tournament!

import random

choices = ["rock", "paper", "scissors"]
player_moves = ["rock", "paper", "scissors", "rock", "paper"]

player_score = 0
computer_score = 0

print("BEST OF 5 TOURNAMENT!")
print("=" * 30)

for round_num, player in enumerate(player_moves, 1):
    computer = random.choice(choices)
    print(f"\nRound {round_num}: You={player}, Computer={computer}")

    if player == computer:
        print("  Tie!")
    elif (player == "rock" and computer == "scissors") or \
         (player == "scissors" and computer == "paper") or \
         (player == "paper" and computer == "rock"):
        print("  You win this round!")
        player_score += 1
    else:
        print("  Computer wins this round!")
        computer_score += 1

print("\n" + "=" * 30)
print(f"FINAL SCORE: You {player_score} - {computer_score} Computer")
if player_score > computer_score:
    print("YOU ARE THE CHAMPION!")
elif computer_score > player_score:
    print("Computer wins the tournament!")
else:
    print("Tournament ends in a tie!")
