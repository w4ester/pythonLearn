Data Structures

PCEP 4.1-4.4Google Week 3

Lists: Ordered Collections

Lists hold multiple items in order. They're mutable (changeable).

# Creating lists
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", True, 3.14]

# Accessing items (0-indexed)
print(fruits[0])   # apple
print(fruits[-1])  # cherry (last item)

# Slicing
print(fruits[0:2])  # ['apple', 'banana']

# Modifying
fruits[0] = "mango"
fruits.append("orange")    # Add to end
fruits.insert(1, "grape")  # Insert at position
fruits.remove("banana")    # Remove by value
popped = fruits.pop()      # Remove & return last

Tuples: Immutable Lists

Like lists but cannot be changed after creation.

# Creating tuples
point = (10, 20)
rgb = (255, 128, 0)
single = (42,)  # Note the comma!

# Accessing (same as lists)
print(point[0])  # 10
x, y = point     # Unpacking

# Tuples are immutable
# point[0] = 5   # Error!

# Good for data that shouldn't change
DAYS = ("Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun")

Dictionaries: Key-Value Pairs

Store data with meaningful keys instead of numeric indexes.

# Creating dictionaries
person = {
    "name": "Alice",
    "age": 25,
    "city": "NYC"
}

# Accessing values
print(person["name"])      # Alice
print(person.get("age"))   # 25
print(person.get("job", "Unknown"))  # Default value

# Modifying
person["age"] = 26         # Update
person["job"] = "Engineer" # Add new key
del person["city"]         # Delete

# Looping
for key, value in person.items():
    print(f"{key}: {value}")

Sets: Unique Items Only

Collections with no duplicates, great for membership testing.

# Creating sets
colors = {"red", "green", "blue"}
nums = set([1, 2, 2, 3, 3, 3])  # {1, 2, 3}

# No duplicates!
colors.add("red")   # Still just 3 items
colors.add("yellow") # Now 4

# Fast membership check
print("red" in colors)  # True

# Set operations
a = {1, 2, 3}
b = {2, 3, 4}
print(a | b)  # Union: {1, 2, 3, 4}
print(a & b)  # Intersection: {2, 3}
print(a - b)  # Difference: {1}

When to Use What?

List

Ordered, changeable, allows duplicates

["a", "b", "c"]

Tuple

Ordered, unchangeable

(1, 2, 3)

Dict

Key-value pairs, fast lookup

{"a": 1}

Set

Unique items, fast membership

{1, 2, 3}
