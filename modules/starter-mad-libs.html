Mad Libs Generator

15 min strings input/output

What We're Building

Remember Mad Libs from road trips? You fill in random words, and they get plugged into a silly story. By the end of this project, you'll have built your own!

Try it first! Click "Run" to see a finished Mad Libs in action. Enter words when prompted! 
# A working Mad Libs - try it!
noun = input("Give me a noun: ")
verb = input("Give me a verb: ")
adjective = input("Give me an adjective: ")
place = input("Give me a place: ")

story = f"""
Once upon a time, a {adjective} {noun}
decided to {verb} all the way to {place}.
Everyone was amazed!
"""
print(story)

Pretty cool, right? Let's build this step by step.

Step 1: Say Hello to Python

First, let's get Python to talk. The print() function displays text on screen.

# Your first Python program!
print("Hello, World!")
print("Welcome to Mad Libs!")
Key concept: Text inside quotes "like this" is called a string. Python treats it as literal text.

Step 2: Storing Words in Variables

A variable is like a labeled box. You put something in, and you can get it back later using the label.

# Creating variables
animal = "dragon"
food = "pizza"
action = "dances"

# Using variables in print
print(animal)
print("The", animal, "loves", food)

Notice: when we type animal without quotes, Python gives us what's inside the box, not the word "animal".

Step 3: Getting User Input

Now for the fun part! input() asks the user for something and stores their answer.

# Ask for input and store it
name = input("What's your name? ")
color = input("What's your favorite color? ")

print("Hello,", name)
print("I like", color, "too!")

The text inside input() is the question that gets shown to the user.

Step 4: Building Sentences (f-strings)

Here's a cleaner way to mix variables with text: f-strings. Put an f before the quotes, then use {variable} to insert values.

# f-strings make it easy!
hero = "Captain Python"
weapon = "keyboard"
enemy = "bugs"

# The 'f' before the string is the magic part
battle_cry = f"{hero} fights {enemy} with a mighty {weapon}!"
print(battle_cry)
Remember: The f stands for "formatted". Without it, you'd literally see {hero} instead of "Captain Python".

Step 5: Your Complete Mad Libs!

Now let's combine everything. This is your Mad Libs generator - run it and have fun!

# Your Mad Libs Generator!
print("=== MAD LIBS: THE SILLY ADVENTURE ===")
print("Fill in these words:\n")

# Get all the words
name = input("Your name: ")
adjective1 = input("An adjective (describes something): ")
noun1 = input("A noun (person/place/thing): ")
verb1 = input("A verb (action word): ")
place = input("A place: ")
adjective2 = input("Another adjective: ")
noun2 = input("Another noun: ")
exclamation = input("An exclamation (Wow! Yikes! Hooray!): ")

# Build the story
story = f"""
THE {adjective1.upper()} ADVENTURE OF {name.upper()}!

One day, {name} found a {adjective1} {noun1}
hiding under the bed. It started to {verb1}!

"Quick! To {place}!" {name} shouted.

But when they got there, a {adjective2} {noun2}
was waiting for them.

"{exclamation}" everyone yelled.

THE END
"""

print(story)

What You Just Learned

  • print("text") - Display text on screen
  • variable = "value" - Store data in a named container
  • input("prompt") - Get text from the user
  • f"Hello {name}" - f-strings for easy text formatting
