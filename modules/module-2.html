Making Decisions with If Statements

Programs need to make decisions. An if statement runs code only when a condition is true.

# Basic if statement
age = 18

if age >= 18:
    print("You can vote!")

# If-else: two paths
if age >= 21:
    print("You can drink alcohol")
else:
    print("No alcohol for you")

# Elif: multiple conditions
score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("F")

Comparison & Logical Operators

# Comparison: ==, !=, <, >, <=, >=
print(5 == 5)   # True
print(5 != 3)   # True
print(5 > 3)    # True

# Logical: and, or, not
age = 25
has_id = True

if age >= 21 and has_id:
    print("Welcome!")

if age < 13 or age > 65:
    print("Discount!")

if not has_id:
    print("ID required")

For Loops

Repeat code for each item in a sequence.

# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# range() generates numbers
for i in range(5):       # 0,1,2,3,4
    print(i)

for i in range(1, 4):    # 1,2,3
    print(i)

for i in range(0, 10, 2): # 0,2,4,6,8
    print(i)

While Loops

Repeat while a condition is true.

# Countdown
count = 5
while count > 0:
    print(count)
    count -= 1
print("Blast off!")

# break exits the loop
for i in range(10):
    if i == 5:
        break
    print(i)  # 0,1,2,3,4

# continue skips to next iteration
for i in range(5):
    if i == 2:
        continue
    print(i)  # 0,1,3,4

List Comprehensions

A shortcut for creating lists:

# Traditional way
squares = []
for x in range(5):
    squares.append(x ** 2)

# List comprehension - same result!
squares = [x ** 2 for x in range(5)]
# [0, 1, 4, 9, 16]

# With condition
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]
