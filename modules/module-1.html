Variables: Storing Information

A variable is like a labeled box that holds a value. You create one by choosing a name and using the = sign.

# Creating variables
name = "Ada"          # A text value (string)
age = 25              # A number (integer)
height = 5.4          # A decimal number (float)
is_student = True     # A true/false value (boolean)

# Using variables
print(name)           # Output: Ada
print("Age:", age)    # Output: Age: 25

Variable Naming Rules

  • Must start with a letter or underscore (_)
  • Can contain letters, numbers, and underscores
  • Case-sensitive: Name and name are different
  • Can't use Python keywords like if, for, class
# Good variable names
user_name = "Python Fan"
totalScore = 100
_private = "hidden"

# Bad variable names (will cause errors!)
# 2fast = "error"      # Can't start with number
# my-var = "error"     # Can't use hyphens
# class = "error"      # Can't use keywords

Operators: Doing Things with Data

Math Operators

# Basic math
a = 10
b = 3

print(a + b)   # Addition: 13
print(a - b)   # Subtraction: 7
print(a * b)   # Multiplication: 30
print(a / b)   # Division: 3.333...
print(a // b)  # Floor division: 3 (rounds down)
print(a % b)   # Modulo (remainder): 1
print(a ** b)  # Exponent: 1000 (10 to the power of 3)

String Operations

# Combining strings (concatenation)
first = "Hello"
second = "World"
greeting = first + " " + second
print(greeting)  # Hello World

# Repeating strings
laugh = "ha" * 3
print(laugh)  # hahaha

# String length
print(len(greeting))  # 11

Input and Output

print() displays output. input() asks the user for input.

# Getting user input
name = input("What's your name? ")
print("Hello, " + name + "!")

# Note: input() always returns a string!
age = input("How old are you? ")
# age is a string like "25", not the number 25

# Convert to number for math
age_number = int(age)
next_year = age_number + 1
print("Next year you'll be", next_year)
